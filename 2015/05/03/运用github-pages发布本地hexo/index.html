设置Github Pages

  1. 进入【your_user_name.github.io】仓库
  2. 点击仓库右侧的setting
  3. 根据需要修改后，点击Automatic page generator
  4. 根据需要修改后，点击continue to layouts
  5. 最后点击publish。
  6. 在浏览器中访问 http://your_user_name.github.io，可以看到github pages
